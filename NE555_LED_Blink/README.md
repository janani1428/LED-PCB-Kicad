# NE555 LED Blinker PCB Design

## Project Overview
This project is a PCB implementation of a reference NE555 Timer based LED Blinker circuit using KiCad. The objective was to understand the complete PCB design workflow, from schematic capture to PCB layout and design verification.

## Design Highlights
- ✅ Schematic Capture
- ✅ Footprint Assignment
- ✅ PCB Layout & Routing
- ✅ 3D PCB Visualization
- ✅ Design Rule Check (DRC) Passed
- ✅ **0 Errors | 0 Warnings**

## Software Used
- KiCad 10

## Components
- NE555 Timer IC
- LED
- Resistors
- Capacitors

## Design Workflow
1. Schematic Design
2. Footprint Assignment
3. PCB Layout
4. Routing
5. Design Rule Check (DRC)
6. 3D PCB Verification

## Learning Outcomes
- Improved understanding of the PCB design workflow in KiCad.
- Practiced schematic capture, component placement, PCB routing, and design verification.
- Successfully completed the design with **DRC: 0 Errors and 0 Warnings**, ensuring the PCB meets the defined design rules.

## Reference
The circuit schematic was recreated from a reference design for educational purposes, while the PCB layout and verification were completed in KiCad as part of my PCB design learning journey.
