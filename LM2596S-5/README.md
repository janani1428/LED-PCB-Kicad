# LM2596-5V Buck Converter PCB Design

## Overview
This project is a PCB design of the LM2596-5V Buck Converter created in KiCad. The design is based on the Texas Instruments LM2596 datasheet reference circuit and was developed for learning PCB design.

## Features
- LM2596-5.0 Fixed 5V Output
- KiCad Schematic Design
- PCB Layout
- 3D PCB Model
- DRC Passed (0 Errors, 0 Warnings)
- Gerber Files Generated

## Tools Used
- KiCad
- Texas Instruments LM2596 Datasheet

## Project Files
- Schematic (.kicad_sch)
- PCB Layout (.kicad_pcb)
- Gerber Files
- Drill Files

## Learning Outcomes
- Reading and understanding a datasheet
- Schematic capture
- PCB layout design
- Component placement
- PCB routing
- Design Rule Check (DRC)
- Gerber generation
